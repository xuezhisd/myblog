发表于 2016-08-28   |   分类于 编程语言 ， Java   |     |   人阅读

java.util.Arrays

Arrays.asList()

  • 数组转换成列表

Arrays.binarySearch()

  • 二分查找
  • 查找前，一定要排序。
  • 如果查找元素不存在，返回(-(insertion point) - 1)。
  • 自然数表示查到，负数表示没有查找。

Arrays.copyOf()

  • 复制长度大于原数组长度时，后面补零。

Arrays.copyOfRange()

  • 复制长度大于原数组长度时，后面补零。
  • Java中区间一般都是左闭右开[a,b)，即包括左边，不包括右边。

Arrays.deepEquals()

  • 比较数组元素是否深层相等。
  • 一维数组无区别，高维数组有区别。

Arrays.deepHashCode()

  • 深层相等的两个数组的深层哈希编码也相等。

Arrays.deepToString()

Arrays.equals()

  • 参考Arrays.deepArrays()

Arrays.fill()

  • 填充数组元素

Arrays.hashCode()

  • 如果两个数组相等，哈希编码也相等。

Arrays.sort()

Arrays.toString()

  • 参考 Arrays.deepToString()
